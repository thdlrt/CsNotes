val kPoints= data.takeSample(false, K, 42).map(s => spark.util.Vector(s))
//takeSample(Boolean, Int, Long)йҮҮж ·еҮҪж•°пјҢfalseиЎЁзӨәдёҚдҪҝз”ЁжӣҝжҚўж–№жі•йҮҮж ·пјҢKиЎЁзӨәж ·жң¬ж•°пјҢ42иЎЁзӨәйҡҸжңәз§Қеӯҗ

 //closestPointи®Ўз®—жңҖиҝ‘зҡ„иҒҡзұ»дёӯеҝғпјҢдә§з”ҹClusterID (spark.util.Vector(p), 1)

    • Spark зҡ„ ML еә“еҹәдәҺDataFrameжҸҗдҫӣй«ҳжҖ§иғҪзҡ„ APIпјҢеё®еҠ©з”ЁжҲ·еҲӣе»әе’ҢдјҳеҢ–е®һз”Ёзҡ„жңәеҷЁеӯҰд№ жөҒж°ҙзәҝпјҢеҢ…жӢ¬зү№еҫҒиҪ¬жҚўзӢ¬жңүзҡ„ Pipelines APIпјҢML жҠҠж•ҙдёӘжңәеҷЁеӯҰд№ зҡ„иҝҮзЁӢжҠҪиұЎжҲҗ PipelineпјҢдёҖдёӘ Pipeline з”ұеӨҡдёӘ Stage з»„жҲҗпјҢжҜҸдёӘ Stage з”ұ Transformer жҲ–иҖ… Estimator з»„жҲҗгҖӮ

    • TransformerпјҡеҸҜд»Ҙе°ҶдёҖдёӘ DataFrame иҪ¬жҚўдёәеҸҰдёҖдёӘ DataFrameгҖӮе®һзҺ° transform ()ж–№жі•гҖӮ
