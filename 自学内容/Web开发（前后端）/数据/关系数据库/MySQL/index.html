      • УАїжћЂтЈфтюетГўтѓет╝ЋТЊјт▒ѓт«ъуј░№╝їУђї MySQL ТюЇтіАтЎет▒ѓТ▓АТюЅт«ъуј░сђѓУАїжћЂТЏ┤жђѓтљѕС║јТюЅтцДжЄЈТїЅу┤бт╝ЋТЮАС╗Хт╣ХтЈЉТЏ┤Тќ░т░ЉжЄЈСИЇтљїТЋ░ТЇ«№╝ї тљїТЌХтЈѕТюЅт╣ХтЈЉТЪЦУ»буџёт║ћуће№╝їтдѓСИђС║Џтюеу║┐С║ІтіАтцёуљє№╝ѕOLTP№╝Ѕу│╗у╗Ъсђѓ

      • тюеТЈљС║цТЋ░ТЇ«ТЏ┤Тќ░С╣ІтЅЇ, Т»ЈСИфС║ІтіАС╝џтЁѕТБђТЪЦтюеУ»ЦС║ІтіАУ»╗тЈќТЋ░ТЇ«тљј, ТюЅТ▓АТюЅтЁХС╗ќС║ІтіАтЈѕС┐«Тћ╣С║єУ»ЦТЋ░ТЇ«сђѓтдѓТъютЁХС╗ќС║ІтіАТюЅТЏ┤Тќ░уџёУ»Ю, ТГБтюеТЈљТў»С║цуџёС║ІтіАС╝џУ┐ЏУАїтЏъТ╗џсђѓ№╝ѕтЇ│жЃйт░ЮУ»ЋС┐«Тћ╣№╝їУДЅтЙЌСИЇС╝џтЄ║жЌ«жбў№╝їтдѓТъютЄ║уј░жЌ«жбўтєЇтЏъжђђ№╝Ѕ
      • С╣љУДѓт╣ХтЈЉТјДтѕХтцџТЋ░ућеС║јТЋ░ТЇ«уФъС║ЅСИЇтцДсђЂтє▓уфЂУЙЃт░Љуџёуј»тбЃСИГ№╝їжђѓтљѕУ»╗ТЊЇСйютцџуџётю║ТЎ»№╝їуЏИт»╣ТЮЦУ»┤тєЎуџёТЊЇСйюТ»ћУЙЃт░Љсђѓ

      • у╝║уѓ╣№╝џТюЅтЈ»УЃйС╝џжЂЄтѕ░СИЇтЈ»жбёТюЪуџёу╗ЊТъю, СЙІтдѓСИцСИфС║ІтіАжЃйтљїТЌХУ»╗тЈќС║єТЋ░ТЇ«т║ЊуџёТЪљСИђУАї, у╗ЈУ┐ЄС┐«Тћ╣С╗ЦтљјтєЎтЏъТЋ░ТЇ«т║Њ, У┐ЎТЌХт░▒жЂЄтѕ░С║єжЌ«жбўсђѓ
