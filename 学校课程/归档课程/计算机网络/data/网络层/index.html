    • IGP (Interior Gateway Protocol): for Intra-AS routing
    • IGPз”ЁдәҺеңЁASпјҲиҮӘжІ»зі»з»ҹпјүеҶ…йғЁиҝӣиЎҢи·Ҝз”ұйҖүжӢ©пјҢе®ғеңЁи·Ҝз”ұеҷЁд№Ӣй—ҙдј йҖ’и·Ҝз”ұдҝЎжҒҜпјҢеҸҜд»Ҙдё“жіЁдәҺжҖ§иғҪпјҢиҖҢдёҚйңҖиҰҒиҖғиҷ‘е…¶д»–ASзҡ„и·Ҝз”ұз®—жі•е’ҢиЎЁж јпјҲеҸҜиғҪдёҚеҗҢпјүгҖӮ
    • RIPжҳҜдёҖз§Қи·қзҰ»еҗ‘йҮҸеҚҸи®®пјҢз”ЁдәҺжөӢйҮҸи·қзҰ»е№¶йҖүжӢ©жңҖзҹӯи·Ҝеҫ„гҖӮ
    • OSPFжҳҜдёҖз§Қй“ҫи·ҜзҠ¶жҖҒеҚҸи®®пјҢе®ғдҪҝз”Ёй“ҫи·ҜзҠ¶жҖҒж•°жҚ®еә“дёӯеӯҳеӮЁзҡ„жӢ“жү‘дҝЎжҒҜжқҘи®Ўз®—жңҖзҹӯи·Ҝеҫ„гҖӮ
    • IGRPжҳҜжҖқз§‘е…¬еҸёдё“жңүзҡ„иҮӘжІ»зі»з»ҹеҶ…йғЁи·Ҝз”ұеҚҸи®®гҖӮ
    • EGP (Exterior Gateway Protocol): for Inter-AS routing
    • EGPз”ЁдәҺеңЁASд№Ӣй—ҙиҝӣиЎҢи·Ҝз”ұйҖүжӢ©пјҢеӣ дёәи·Ҝз”ұеҷЁйңҖиҰҒдәҶи§Је…¶ASд№ӢеӨ–зҡ„зҪ‘з»ңжғ…еҶөпјҢе®ғж”ҜжҢҒе…ідәҺеҸҜиҫҫжҖ§зҡ„ж‘ҳиҰҒдҝЎжҒҜгҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢзӯ–з•ҘеҸҜиғҪдјҡдјҳе…ҲдәҺжҖ§иғҪгҖӮпјҲиҰҒе…је®№пјү
    • BGPжҳҜдёҖз§Қиҫ№з•ҢзҪ‘е…іеҚҸи®®пјҢз”ЁдәҺеңЁдёҚеҗҢиҮӘжІ»зі»з»ҹд№Ӣй—ҙдј иҫ“и·Ҝз”ұдҝЎжҒҜгҖӮ
    • EGPиҰҒжұӮзҡ„зү№жҖ§пјҡ

    • и·қзҰ»еҗ‘йҮҸеҚҸи®®ж— жі•жҸҗдҫӣжңүе…із»ҸиҝҮзҡ„ASзҡ„дҝЎжҒҜпјӣ
